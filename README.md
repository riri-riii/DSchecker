# wlw DS値計算機

wlwのDS値計算サイトです。

---

## DS値計算機

キャストとアシスト・ソウルを選択すると、レベルごとのDS値と1確までの残り値を自動で計算します。

### 使い方

1. **キャスト選択** でキャストを選びます
2. **アシスト1〜4・ソウル** の入力欄にアシスト名やよみがなを入力すると候補が表示されるので、選んでください
3. 各レベルのDS値と「1確残」が自動で計算されます

### 補足

- アシスト4には、Lv6以上のアシストのみ表示されます
- 値はキャスト専用効果など、特殊条件がすべて乗った合計値です
- XDSを使用する場合は **127.82 を加算** して計算してください（投擲の場合は85）
- 現在キャストは**ファイターおよびマリアン**のみ掲載しています

## DS値検証記録

検証結果を閲覧できるページです。
作成者から提供される編集用コードを入力することで、結果の投稿ができるようになります。
（詳細は下部「記録・削除（ご協力者様向け）」をご覧ください）

### 記録の見方

```
検証アシスト名[キャスト_使用アシスト構成]
レベルN(DS値)：1確残  ○ / ×
```

- **○** … そのレベルで兵士を取れた（自由曲線の場合は8体）
- **×** … 取れなかった
- **1確残** … 1確ラインとのDS値の差（＋なら余裕あり、－なら不足）

### 

検証結果の投稿・削除には **編集用コード** が必要です。

1. 右上の「ご協力者様はこちら」ボタンをクリックしてください
2. 作成者から共有された編集用コードを入力してください
3. 認証が通ると、検証情報の入力と記録の削除ができるようになります

認証は30日間有効です。期限が切れた場合は再度コードを入力してください。

#### 記録の手順

1. 「検証アシスト名」に検証しているアシストの名前を入力します
2. キャスト・アシスト・ソウルを選択します
3. 各レベルで実際に検証し、結果の「結果」列に取れた場合はチェックを入れます
4. ログに残したいレベルの「記録」列にチェックを入れ、「記録」ボタンを押します
5. 必要であれば補足情報を入力して保存してください
